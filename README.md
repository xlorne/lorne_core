
### 框架介绍V1.0.0：

  该框架属于工具类底层框架。停止维护，新框架见https://github.com/codingapi/springboot-framework
  
 
### 框架支持功能如下：

1. 整合了apache http框架  
2. 封装了AES Base64 Md5加密算法处理类  
3. 封装了Task线程等待唤醒机制  
4. 封装了Thread本地多线程同步数据汇总框架  
5. 封装了本地配置资源文件读写    
6. 封装了DateUtils MathUtils KidUtils JsonUtils 常用工具类封装
